=================================================
SMART RESUME REVIEWER - SETUP & RUN INSTRUCTIONS
=================================================

1. PREPARE THE PROJECT
   Ensure your project folder contains:
   - main.py (and the 'agents', 'utils' folders)
   - requirements.txt

2. OPEN COMMAND PROMPT
   Copy the path of your project folder.
   Open cmd and type:
   cd C:\smart-resume-reviewer
   (Replace with your actual path)

3. ACTIVATE VIRTUAL ENVIRONMENT
   Run this command:
   venv\Scripts\activate

4. INSTALL DEPENDENCIES
   Run this command to install all necessary libraries:
   pip install -r requirements.txt

5. RUN THE APP
   Execute the following command:
   streamlit run main.py

6. USE THE APP
   - A browser window will open automatically.
   - Enter a Job Title (e.g., "Python Developer").
   - Upload your Resume PDF.
   - The tool will analyze the match score and give feedback!
